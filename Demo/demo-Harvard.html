TinyMCE v7 Harvard Citation Plugin

A powerful citation plugin that generates Harvard style citations with hover tooltips. Perfect for academic writing and research papers.

How to Use

  • Type @ to trigger the citation menu
  • Select a source from the dropdown
  • A short citation (Author, Year) will be inserted
  • Hover over any citation to see the full Harvard style format
  • Citations are automatically formatted based on their source type following Harvard guidelines
